# Gzip Compress/Decompress

Compress text with gzip or decompress gzip data online. Supports base64 and hex output formats. Free, fast, and privacy-first.

## Features

- Compress text to gzip and get raw bytes, Base64, or hex
- Decompress gzip from raw, Base64, or hex input
- Auto-detect compress vs decompress when possible
- Useful for HTTP payloads, logs, or small archives
- All processing in browser; no data sent to server

## Why use this tool

Inspect gzip-compressed API responses or logs, or produce compressed payloads for testing. Handles the encoding formats you actually see.
